# Weather App

A simple and elegant weather application built with Flutter and powered by the OpenWeatherMap API. This app provides current weather information and a 5-day forecast for any city in the world.

## Features

- Current weather information
- 5-day weather forecast
- Settings to change the city
- Responsive and modern UI design
